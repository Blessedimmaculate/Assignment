Paradigm shift and Proactivity

A paradigm shift refers to a fundamental change in one's mindset, perspective, or approach to a particular aspect of life.
Proactivity, on the other hand, refers to the ability to take initiative and take control of one's life, circumstances, and choices.
1. Asking empowering questions
2. Setting clear goals and intentions
3. Surrounding yourself with positive influences
4. Practicing mindfulness and self-reflection
5. Embracing lifelong learning and personal development

GITHUB, CSS AND BOOTSTRAP

- CSS (Cascading Style Sheets) is used for styling and layout purposes, controlling the visual aspects of a website or web application.
- GitHub is a web-based platform for version control and collaboration on software development projects, allowing developers to store, manage, and share their code.
- Bootstrap is a popular front-end framework used for building responsive, mobile-first websites and web applications, providing a set of pre-designed UI components and a robust grid system.
